DecimalPipe y Porcentaje

Pipe para 'mostrar' decimales incluidos en el Common Module

{{ ventasNetas | number:'1.2-2'}}     /     {{ (ventasNetas) * 0.99 | number:'1.2-2' }}

{{ ventasBrutas | currency:'EUR' }}    /     {{ (ventasBrutas + 2000000) * 0.99 | currency:'USD' }}

{{ ((ventasNetas) / (ventasBrutas))*100 | percent:'2.2-2' }}
